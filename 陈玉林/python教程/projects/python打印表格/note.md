## mytable

### 功能
打印漂亮的表格

### 注意事项
打印的结果只能在等宽字体下才能正常显示,且中文字符的宽度需要是半角字符的两倍

在非终端界面,渲染的宽度似乎不是二倍的关系(目测小于二倍,没有找到相关资料),所以显示会有错位.
